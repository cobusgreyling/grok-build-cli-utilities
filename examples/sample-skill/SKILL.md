# Sample Utility Skill

This is a minimal but complete example skill you can unpack with:

    grok-utils skills unpack examples/sample-skill.skill.tar.gz

## Steps

1. Greet the user enthusiastically.
2. Run `grok-utils --version` to demonstrate calling the utilities.
3. Report success.
