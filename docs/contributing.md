# Contributing

Please read the full [CONTRIBUTING.md](https://github.com/cobusgreyling/grok-build-cli-utilities/blob/main/CONTRIBUTING.md) in the repository root.

## Quick summary

- This is a **maintainer-driven** project (sole author Cobus Greyling).
- Every commit so far is by the maintainer. Drive-by PRs without prior discussion are unlikely to be merged.
- High-quality, well-tested, focused contributions that respect the "safe by default", beautiful output, and minimal-dep philosophy are welcome.
- Read CONTRIBUTING.md for testing expectations, code style, PR process and what is in-scope.
- For large ideas, open an issue first and discuss.

Thank you for helping make Grok Build even more productive!
